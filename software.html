1. System Architecture

Core computing (IK, Vision, VLA Policy inference) runs either on your Host Laptop (tethered) or the onboard NVIDIA Jetson Orin Nano. The architecture utilizes a direct USB-serial connection to three independent motor buses.

┌──────────────┐                  ┌────────────────────────┐

│ (Follower    │◄────────────────►│                        │

2. Framework Installation

We recommend using Ubuntu 22.04 and Python 3.10. The XLeRobot-Pro repository contains the integrated LeRobot framework and hardware-specific configurations.

Clone & Install

Terminal
# Clone the unified pro repository
$ git clone https://github.com/Minko82/xlerobot-pro
$ cd xlerobot-pro

# Install dependencies in editable mode
$ pip install -e .
JETSON USERS: Ensure you have installed JetPack 5.1.x or 6.0. If using Orin Nano, we recommend enabling "Max Power Mode" via sudo nvpmodel -m 0 for policy inference.

3. Environment & Port Configuration

A. USB Serial Permissions

Grant read/write access to the motor controller ports. These usually appear as ttyACM devices.

Grant Permissions
$ sudo chmod 666 /dev/ttyACM*

# Verify your connected ports:
$ python find_port.py

B. Camera Verification

Check if the three USB cameras (2x Wrist, 1x Head) are detected by the OS.

V4L2 Check
$ v4l2-ctl --list-devices
